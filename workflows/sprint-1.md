
# Sprint 1

## Goal

Build a desktop-first command center that gives John the daily priorities and automates Craven marketing first.

## Features

- Executive Command Center
- AI Marketing Center
- Craven Off-Season Dashboard
- W&D Bid Center
- AI Assistant panel

## Success Criteria

John can open the application and immediately see:
- What must be done today
- Which marketing package is ready
- Which bids need attention
- Which business dashboard to open next
