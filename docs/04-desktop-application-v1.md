
# Desktop Application v1

## Design Decision

Portillo AI will be desktop-first.

## Reason

John performs high-value work on desktop:
- Reviewing bid documents
- Building proposals
- Managing email
- Creating marketing packages
- Organizing documents

Mobile will be built later as a companion for quick approvals, notifications, and voice commands.
