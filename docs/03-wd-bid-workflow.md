# W&D Portico Bid Workflow

## Current Workflow

1. Receive email invitation to bid.
2. Click link to review bid documents in BidConnect.
3. Open document file.
4. Download plans.
5. Upload plans to ChatGPT.
6. Request bid draft.
7. Review quantities and specifications.
8. Go back and forth until the bid matches W&D Portico style.
9. Download proposal.
10. Upload proposal to BidConnect using the bid form.
11. Receive notification when GC views bid.
12. Provide additional information if requested.
13. Know the bid is won when GC contacts John.

## Portillo AI Goal

Create a Bid Lifecycle Manager that tracks every bid from invitation to award.
